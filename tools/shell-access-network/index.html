2014-08-04 00:00:00 |

shell中访问网络工具curl、wget用法

curl

发送cookie

curl -b “ver=1;help=2;” www.xlq.com/test.php

post方式传递参数

curl -d “ver=1&help=2” www.xlq.com/test.php

伪造http头

curl -H “User-Agent:Mozilla/5.0 (Windows NT 5.1)” -H “cookie:ver=1;” www.xlq.com/test.php

wget

断点下载

wget -c -t 0 [url]
