携带的cookie

字段 备注

hybrid.js源代码：

(function () {
    function getArguments(args) {
        var result = [];
        for (var key in args) {
            result.push(args[key]);
        }
        return result;
    }

    window.AppLoad = function (callback) {
        if (window.WebViewJavascriptBridge) {
            callback(WebViewJavascriptBridge)
        } else {
            document.addEventListener('WebViewJavascriptBridgeReady', function () {
                    callback(WebViewJavascriptBridge)
                },
                false
            );
        }
    };
    window.AppShare = function (title, content, url, img) {
        window.WebViewJavascriptBridge.callHandler(
            'share',
            getArguments(arguments),
            null
        );
    };
})();

发送消息

向android端需要发送自定义消息，消息体如下：

字段 备注
title 消息标题，20个字以内为宜
content 消息内容，50个字以内为宜
icon 图标
url 消息链接

服务器端php参考代码：

$client = new JPush($appKey, $masterSecret);

$message = array(
    'title'   => '测试标题',
    'content' => '测试内容',
    'url'     => 'http://www.baidu.com',
    'icon'    => 'http://q2.qlogo.cn/g?b=qq&k=dXUibRnYqBTncxGMHic5hOsA&s=40&t=1451886050',
);
$result = $client->push()
    ->setPlatform('all')
    ->addRegistrationId('120c83f760247174b6d')
    ->setMessage(json_encode($message))
    ->send();
