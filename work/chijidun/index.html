首页私厨数据拉取细节

下面例子中除perPage、pos_lat、pos_lng不固定，其他参数都应该是固定的
type:0、1、2分别对应第1、2、3段数据

获取配送范围内上班的私厨

curl http://218.244.130.96/chu/chiApi/member/lists \
    -d 'data={\
        "perPage":1000,\
        "pos_lat":30.294565,\
        "pos_lng":120.003241,\
        "type":0\
     }'

获取配送范围内下班的私厨

curl http://218.244.130.96/chu/chiApi/member/lists \
    -d 'data={\
        "perPage":1000,\
        "pos_lat":30.294565,\
        "pos_lng":120.003241,\   
        "type":1\
     }'

获取配送范围外下班的私厨

curl http://218.244.130.96/chu/chiApi/member/lists \
    -d 'data={\
        "perPage":1000,\
        "pos_lat":30.294565,\
        "pos_lng":120.003241,\
        "type":2\
     }'
